ARRAYS

Arrays are ordered, integer-indexed collections of any object. Basically arrays are like containers for data. Arrays are created by giving your array a name and setting equal to a value with brackets [ ]. The values are separated by commas and like always strings must have quotation " " marks around them.

array = [2, 4, 6, 8, 10]

Arrays are sorted by the rank of value by default and the value of each array starts at 0. In the array above, the value of 2 would have the index of [0] and 4 would have an index of [1] and so on. Also the values in the array can be called in backwards order with negative numbers. For example, the value of 10 would have an index of [-1] and 8 would have index of [-2]. We would use the code “array[3]” to call the value 8.

HASHES

Hashes are also another way of storing data. They are similar to arrays, but instead of storing data based on numeric indices, you use a collection of key-value pairs. Values are assigned to keys using the " => " sign. The hash syntax looks like this:

hash = {
key1 => value1,
key2 => value2,
key3 => value3
}

me = {
name = "Michael"
state = "California"
age = 24
}

If we want to access the key-value pair and puts the value of my name, we would type " puts me["name"] ".
