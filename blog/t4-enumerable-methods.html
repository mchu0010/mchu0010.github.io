.map Method

.map is a Ruby method that takes a block and applies the expression in the block to every element in the array. For example:

ary = [1,2,3,4]
ary.map {|x| x*2 }
# => [2,4,6,8]

If we look at a value of ary, weâ€™ll see the that the value has not changed:

array
# => [1,2,3,4]

As you can see, the value of ary did not change because the .map method returns a copy of ary, but it does not change the original ary array. If we want to change the original array, we can use add the exclamation point to the end of map that that would then look like .map!

ary.map! {|x| x*2}
# => [2,4,6,8]
arr
# => [2,4,6,8]

One last thing about the map method. The .collect method is identical to the map method, so either one can be used.
