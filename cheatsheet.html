Basic Git Commands Cheatsheet

Git Commands

git status:
Lists all new or modified files

git branch:
Lists all local branches in the current repository

git checkout:
Switches between branches

git add:
Stages the file to be committed

git fetch:
Downloads from the repository

git clone:
Copies an existing repository

git push:
Uploads all local branch commits to GitHub

git commit -m:
Records file snapshots permanently in version history

git merge:
Combines the specified branch's history into the current branch

git log:
Lists version history for the current branch

git rm:
Deletes the file from the working directory

git reset:
Undoes all commits afer the commit
